Laurence's Painting Laboratory

This laboratory is WebGL-based and contains two parts.

Part 1: A model showroom

In this page you can enjoy several models in the application. By using controllers in the right hand, you could change the appearance of those models. The image will navigate you to the page.

Part 2: A WebGL Demo

In this page we create a WebGL Demo using models and shaders in the previous part. The image will navigate you to the page.

CAUTION

Before you access to those pages, please be sure that your browser does support WebGL.

Many browser's latest version support WebGL, as the following listed:

  • Google Chrome
  • Mozilla Firefox
  • Apple Safari
  • Opera Mini
